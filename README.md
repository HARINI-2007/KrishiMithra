# 🌱 KrishiMithra

KrishiMithra is an AI-powered smart agriculture application that helps farmers identify plant diseases from leaf images and provides treatment recommendations in a simple, user-friendly way.

## 🚀 Features

- 🌿 Plant disease detection using AI
- 📷 Upload leaf images
- 🔍 Disease prediction
- 💊 Recommended fungicides/pesticides
- 📖 Disease description and symptoms
- 📱 Attractive and responsive UI
- 🌾 Farmer-friendly interface

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript
- HTML
- CSS
- AI/ML Model
- VS Code

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/HARINI-2007/KrishiMithra.git
```

Go to the project folder:

```bash
cd KrishiMithra
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
